# Tomorodo Timer

A clean, minimal Pomodoro timer built with vanilla HTML, CSS, and JavaScript.

## Features

- **Three modes** — Focus (25 min), Short Break (5 min), Long Break (15 min)
- **Circular progress ring** — animated SVG ring that drains as time counts down, with a color-matched glow
- **Session tracker** — 4-dot indicator that tracks your focus sessions through a full Pomodoro cycle
- **Auto mode switching** — automatically moves from Focus → Short Break after each session, and resets after 4 sessions
- **Skip button** — jump to the next mode at any time
- **Mobile responsive** — scales cleanly on small screens

## Usage

No build step or dependencies. Just open `index.html` in a browser.

## Tech

- HTML5 / CSS3 / Vanilla JS
- SVG stroke animation for the progress ring
- CSS custom properties for theme switching between modes
